# Портфолио проектов

Репозиторий содержит мои учебные и личные проекты, демонстрирующие навыки в области **программирования на C++** и **DevOps-практик**.

---

## Структура репозитория

RepoWithProjects/
├── cpp-projects/ # Проекты на C++
│ ├── readme.md # Описание проектов
│ └── ... # Директории с исходным кодом
├── devops-projects/ # Инструменты сборки, контейнеризации и CI
│ ├── readme.md # Описание инфраструктурных решений
│ └── ... # Директории с исходным кодом
└── README.md # **Вы здесь**

---

## Основные компетенции

- **C++ (17/20/23)**: наследование, итераторы, аллокаторы, move-семантика, умные указатели, SFINAE, концепты
- **Сборка и тестирование**: CMake (многоуровневая вложенность, code coverage, линтеры), GoogleTest, GoogleMock
- **DevOps**: Linux, Docker Compose, настройка CI(GitHub Actions)

---

## Как смотреть проекты

Перейдите в соответствующую папку и изучите `README.md` внутри неё - там даны инструкции по сборке, запуску тестов и описание ключевых особенностей.
