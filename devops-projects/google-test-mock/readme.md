# C++ Testing Project

Набор задач по тестированию на C++ с использованием **Google Test**, **Google Mock** и анализом покрытия кода.

---

## Технологический стек

| Технология        | Назначение                           |
|-------------------|--------------------------------------|
| C++17             | Язык программирования                |
| CMake             | Система сборки                       |
| Google Test       | Фреймворк для unit-тестирования      |
| Google Mock       | Создание mock-объектов               |
| CPR               | HTTP-запросы (для работы с API)      |
| nlohmann/json     | Парсинг JSON                         |
| lcov / gcov       | Анализ покрытия кода                 |

---

## Описание задач

### 1. `00-aplusb` — Простая функция

- Функция `Add(a, b)` — сложение двух чисел
- Тесты для проверки работы с разными аргументами
- Использование параметризованных тестов

### 2. `01-simple-library` — Календарные функции

- Проверка високосного года
- Получение количества дней в месяце
- Тесты с фикстурами и параметризацией

### 3. `02-tree` — Файловая система

- Обход директорий и построение дерева файлов
- Фильтрация пустых директорий
- Работа с `std::filesystem`

### 4. `03-weather` — HTTP-запросы и моки

- Интеграция с AccuWeather API
- Мокирование HTTP-запросов через Google Mock
- Тестирование без реальных сетевых вызовов
- Использование JSON-заглушек

## Сборка и запуск

```bash
mkdir build
cmake -S . -B build -DCMAKE_BUILD_TYPE=debug
cmake --build build/ --target coverage_report -j16
bin/00-aplusb-test
bin/01-simple-library-test 
bin/02-tree-test
bin/03-weather-test
```

---

## Отчет

В директории build появятся следующие папки:

- 00-coverage
- 01-coverage
- 02-coverage
- 03-coverage
В каждой из них лежит файл index.html, который можно открыть в браузере для получения результатов покрытия.
