# DevOps и инфраструктурные проекты

В этой директории собраны проекты, демонстрирующие владение инструментами промышленной разработки: системы сборки, контейнеризация, CI/CD, тестирование и анализ покрытия.

---

## Состав

| Проект | Краткое описание |
|--------|-------------------|
| [`train-your-dragon`](./train-your-dragon) | Сборка многомодульного C/C++ кода с помощью **CMake**, GoogleTest, линтингом, генерацией покрытия и профилями сборки |
| [`fullstack-webapp`](./fullstack-webapp) | Трёхзвенное веб-приложение (**FastAPI + PostgreSQL + Nginx**), развёрнутое через Docker Compose с инициализацией БД и healthcheck’ами |
| [`google-test-mock`](./google-test-mock) | Задачи по модульному тестированию C++ с использованием **Google Test**, **Google Mock**, мокированием HTTP-запросов и анализом покрытия |
| [`ci-pipeline`](./ci-pipeline) | Настройка **GitLab CI** для сборки, тестирования и доставки C++ приложения с подключением Google Test |

---

## Ключевые навыки

* **Сборка и тестирование**: CMake (многоуровневые проекты, внешние модули), GoogleTest, GoogleMock
* **Контейнеризация и оркестрация**: Docker, Docker Compose, сети и тома
* **CI/CD**: GitLab CI, автоматическая сборка, тестирование и артефакты
* **Качество кода**: линтинг (include-путей), покрытие кода (lcov/genhtml)
* **Сети и веб**: Nginx, FastAPI, работа с HTTP и JSON

Подробное описание, технологический стек и инструкции по запуску находятся в `readme.md` каждого проекта.
