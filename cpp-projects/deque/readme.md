# Deque

Реализован шаблонный класс `Deque<T>` - аналог класса `std::deque<T>`.

`Deque` обладает следующей функциональностью:

- Конструкторы:
  - По умолчанию (не должен совершать аллокаций)
  - Копирования
  - От `int`
  - От `int` и `const T&`
- Оператор присваивания.
- Метод `Size()`, возвращающий текущий размер контейнера.
- Обращение по индексу:
  - `operator[]` (без проверок выхода за границу)
  - `At` (кидающее `std::out_of_range`)

    работающее за гарантированное `O(1)`.
- Метод `Swap`, принимающий `Deque` и меняющий местами содержимое текущего контейнера и параметра. Работает за гарантированное `O(1)`.
- Внутренний тип `iterator`. Этот тип поддерживает:
  - Инкремент, декремент
  - Сложение с целыми числами
  - Операторы сравнения
  - Взятие разности двух итераторов
  - `operator*`
  - `operator->`
- Также реализован константный итератор `const_iterator` так, что его код не является копипастой обычного `iterator`.
- Методы `PushBack`, `PopBack`, `PushFront`, `PopFront`, работающие за амортизированное `O(1)`.

    **Операции `PushBack`, `PushFront`, `PopBack` и `PopFront` не инвалидируют указатели и ссылки на остальные элементы дека. Операции `PopBack` и `PopFront`, кроме того, еще и не инвалидируют итераторы.**
- Методы `begin`, `cbegin`, `end` и `cend`, возвращающие неконстантные и константные итераторы на начало и на “элемент после конца” контейнера соответственно.
- reverse-итераторы, а также методы `rbegin`, `rend`, `crbegin`, `crend`.
- Метод `Insert(const_iterator, const T&)`, делающий вставку в контейнер по итератору за линейное время в среднем.
- Метод `Erase(const_iterator)`, удаляющий элемент из контейнера по итератору за линейное время в среднем.

    **Кроме того, `Insert` и `Erase` по итератору на начало или конец дека не инвалидируют указатели и ссылки на остальные элементы дека.**
- **Все методы `Deque` (кроме `Insert` и `Erase` в место, отличное от начала и конца контейнера) строго безопасны относительно исключений.**

### Запуск тестов

осуществляется следующими командами:

```bash
cd cpp-projects && mkdir build
cmake -S . -B build
cmake --build build/ --target test_deque && build/test_deque
```
