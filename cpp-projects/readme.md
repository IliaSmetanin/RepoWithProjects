# Проекты на C++

Реализация ключевых компонентов стандартной библиотеки и обобщённых интерфейсов на C++ (стандарты 17/20/23).  
Все проекты собираются через **CMake**, тестируются с помощью **GoogleTest**.

---

## Состав

| Проект | Краткое описание |
| -------- | ------------------- |
| `biginteger` | Целые числа сколь угодной величины с перегрузкой стандартных операций |
| `rational` | Рациональные числа сколь угодной точности с использованием biginteger |
| `geometry` | Полиморфная иерархия классов для работы с фигурами на плоскости |
| `deque` | Двусторонняя очередь с кастомными итераторами и строгой гарантией безопасности исключений |
| `list` | Двусвязный список с кастомным пул-аллокатором, итераторами и поддержкой move-семантики |
| `unordered-map` | Хеш-таблица на базе собственного `list` с поддержкой move-семантики |
| `smart-pointers` | Умные указатели (`SharedPtr`, `WeakPtr`, `EnableSharedFromThis`) с разделяемым владением и поддержкой идиомы type erasure |
| `tuple` | Обобщённый кортеж с использованием концептов и SFINAE |
| `function` | Аналог `std::function` и `std::move_only_function` с type erasure |

Подробности реализации, инструкции по сборке и примеры использования — в `README.md` внутри каждой папки.
