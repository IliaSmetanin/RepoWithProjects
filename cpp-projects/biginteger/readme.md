# BigInteger

Представлен класс `BigInteger` для работы с длинными целыми числами, в котором поддерживаются операции:

- Сложение, вычитание, умножение, деление, остаток по модулю, работающие так же, как и для `int`; составное присваивание с этими операциями. Умножение, деление и остаток по модулю работают не дольше, чем за $O(n^2)$.
- Унарный плюс, унарный минус, префиксные и постфиксные инкремент и декремент. Префиксный инкремент и декремент работают за $O(1)$ в среднем, а также не делают избыточных аллокаций.
- Оператор `<=>`, сравнения работают с его помощью.
- Оператор `==`.
- Вывод в поток (корректно обрабатывающий и числа с ведущими нулями) и ввод из потока.
- Метод `ToString()`, возвращающий строковое представление числа.
- Конструирование из `int64_t`, `uint64_t` (в том числе неявное преобразование), `std::string` (из любой строки из цифр, в частности, допустимы ведущие нули).
- Преобразование в `bool` (должно работать в условных выражениях).
- Литеральный суффикс `_bi` для написания литералов типа `BigInteger`. Работает как от чисел (вида `37183123127313234632762740326741_bi`), так и от C-style строк (вида `"2346327624203647324"_bi`).

## Запуск тестов

осуществляется следующими командами:

```bash
cd cpp-projects && mkdir build
cmake -S . -B build
cmake --build build/ --target test_biginteger && build/test_biginteger
```
