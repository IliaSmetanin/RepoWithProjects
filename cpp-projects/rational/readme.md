# Rational

С использованием класса `BigInteger` из biginteger/ был написан класс `Rational` для работы с рациональными числами сколь угодно высокой точности. Числа `Rational` представляются в виде несократимых обыкновенных дробей, где числитель и знаменатель – сколь угодно длинные целые числа. Поддерживаются следующие операции:

- Конструктор из `BigInteger`, из двух `BigInteger` (числитель и знаменатель), из `int64_t`, из `uint64_t`.
- Ввод из потока в формате `[-]numerator/denominator` или `[-]integer.fractional`. При вводе пользователь может вводить целые числа и как `42/1`, и как `42`, и как `42.0`, и как `42.000`, все эти способы должны корректно работать.
- Вывод в поток в формате `[-]numerator/denominator`.
- Сложение, вычитание, умножение, деление, составное присваивание с этими операциями.
- Унарный плюс, унарный минус.
- Операторы сравнения `==`, `<=>`.
- Метод `ToString()`, возвращающий строковое представление числа (вида `[-]numerator/denominator`, где числитель и знаменатель - взаимно простые числа; если число целое, то знаменатель не выводится).
- Метод `AsDecimal(size_t precision=1)`, возвращающий строковое представление числа в виде десятичной дроби с $precision$ знаками после запятой.
- Оператор приведения к `double` (допустимая точность $10^{-9}$).
- Литеральный суффикс `_rat` для написания литералов типа `Rational`. Он работает как от чисел, так и от C-style строк. C-style строки могут быть в том же виде, что и для ввода из потока.

### Запуск тестов

осуществляется следующими командами:

```bash
cd cpp-projects && mkdir build
cmake -S . -B build
cmake --build build/ --target test_rational && build/test_rational
```
